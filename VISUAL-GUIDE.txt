┌──────────────────────────────────────────────────────────────┐
│  🔐 AUTHENTICATION FLOW                                      │
└──────────────────────────────────────────────────────────────┘

REGISTER:
  User → Frontend → Backend → Hash Password → Save to DB
                            → Generate JWT → Return Token

LOGIN:
  User → Frontend → Backend → Verify Password → Generate JWT
                            → Return Token → Store in localStorage

PROTECTED REQUESTS:
  Frontend → Add Token to Header → Backend Middleware
          → Verify Token → Allow Access to User's Data

┌──────────────────────────────────────────────────────────────┐
│  📊 API ENDPOINTS                                            │
└──────────────────────────────────────────────────────────────┘

PUBLIC:
  POST /api/auth/register    ← Create account
  POST /api/auth/login       ← Login

PROTECTED (Requires Token):
  GET  /api/auth/me          ← Current user
  GET  /api/orders           ← User's orders
  POST /api/orders/bulk      ← Bulk create orders
  GET  /api/customers        ← User's customers
  POST /api/customers/bulk   ← Bulk create customers
  GET  /api/tasks            ← User's tasks
  POST /api/tasks            ← Create task
  PUT  /api/tasks/:id        ← Update task
  DELETE /api/tasks/:id      ← Delete task

┌──────────────────────────────────────────────────────────────┐
│  🐛 TROUBLESHOOTING                                          │
└──────────────────────────────────────────────────────────────┘

MongoDB not connecting?
  → Check MongoDB is running
  → Verify .env connection string

CORS errors?
  → Ensure backend is on port 5000
  → Check api-config.js URL

Token errors?
  → Logout and login again
  → Clear localStorage

Can't install packages?
  → Delete node_modules folder
  → Run npm install again
